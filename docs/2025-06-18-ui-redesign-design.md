# 海龟汤网站 UI 重新设计

## 概述

在保留侦探档案/纸张质感主题的基础上，对海龟汤网站进行彻底的视觉升级，加入动态粒子效果、精致动画、更干净的布局，全面提升用户体验。

## 动态效果

### 1. Canvas 粒子系统
- 新建 `static/js/particles.js`，纯 JS canvas 实现
- 30-40 个半透明粒子（小圆点），颜色在 `#C49B5C`、`#E8DFD0`、`#8B2635` 柔和变化
- 粒子在背景层缓慢上下飘浮（sin 波运动）
- 鼠标移动时粒子受"推开"力影响，营造拨开烟雾的效果
- 粒子透明度 0.15-0.35，不干扰内容阅读
- canvas 固定定位在全屏背景

### 2. 动态光晕
- CSS `::before`/`::after` 伪元素在 `body` 或 `.container` 上
- 使用 `background: radial-gradient()` + `@keyframes` 动画
- 两个光晕（金色 `#C49B5C` 和柔和红 `#8B2635`）缓慢漂移
- `animation: drift 20s ease-in-out infinite`

### 3. 卡片进场动画
- 使用 `IntersectionObserver` 实现
- 故事卡片进入视口时从下方滑入 + 淡入
- 每张卡延迟 `index * 80ms`
- 过渡属性：`opacity 0→1, transform: translateY(30px)→translateY(0)`

### 4. 打字机效果
- 进入故事页面时，"汤面"内容逐字显示
- 每个字间隔 30ms，用 `setInterval` 实现
- AI 回答也使用打字机效果（50ms/字）

### 5. 微交互质感
- 卡片 hover：`translateY(-4px)` + 阴影加深 + 轻微旋转 (0.5deg)
- 按钮 hover：颜色加深 + 1px 下移 `translateY(1px)`
- 筛选标签 active：金色边框 + 金色底色填充
- 所有过渡统一 `0.2s ease`

## 文件变更清单

### 新增文件
1. `static/js/particles.js` — Canvas 粒子系统
2. `static/js/typewriter.js` — 打字机效果工具函数

### 修改文件
1. `static/index.html` — 引入新的 JS 文件，添加 canvas 元素
2. `static/css/style.css` — 全面重新设计样式
3. `static/js/app.js` — 集成粒子、打字机、进场动画

## 实现顺序

1. 先实现 CSS 全面重新设计（配色、布局、质感）
2. 实现 `particles.js` 并集成
3. 实现 `typewriter.js` 打字机效果并集成
4. 实现卡片进场动画（在 app.js 中）
5. 优化微交互
6. 测试和调整
